# P2PMessenger 

P2P-мессенджер для iOS

## Быстрый старт

### Автоматически
1. Клонируйте репозиторий.
2. В папке проекта найдите файл **`setup.command`**.
3. Дважды кликните по нему. 

Скрипт сам проверит наличие необходимых инструментов и сгенерирует актуальный `.xcodeproj`.

### Через терминал:
Если вы предпочитаете командную строку:
```bash
make -f .github/Makefile setup
```
