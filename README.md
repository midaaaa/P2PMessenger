# P2PMessenger 

P2P-мессенджер для iOS

## Быстрый старт

### Автоматически
1. Клонируйте репозиторий.
2. В папке проекта найдите файл **`setup.command`**.
3. Дважды кликните по нему. 

Скрипт сам, при первом запуске спросит ваш префикс и создаст файл `local.yml`, проверит наличие необходимых инструментов и сгенерирует актуальный `.xcodeproj`.

### Через терминал:
1. Клонируйте репозиторий.
2. Откройте терминал в папке проекта.
3. Выполните команду:
```bash
make -f .github/Makefile setup
```
### Изменение bundleIdPrefix
Если хотите изменить префикс позже:
1. Создайте/найдите в корне проекта файл `local.yml`:
```yaml
options:
  bundleIdPrefix: com.yourname
```
2. Выполните команду:
```bash
make -f .github/Makefile setup
```

## Работа с локализацией

Многие текстовые поля вида `Text("...")` добавляются в каталог автоматически. Для добавления строк в другие конструкции использовать `String(localized: "...")`.

Если необходимо добавить переменную из кода в такую строку можно использовать 
```swift
LocalizedStringResource("ключ", defaultValue: "\(переменная)")
```
или
```swift
String(format: String(localized: "ключ"), переменная)
```

Ключи решили называть в формате *snake_case* или *lowerCamelCase*.
